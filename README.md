# Elm Prague Coding Dojo #6

## Run Tests

```sh
yarn test:watch
```

Or `yarn test` to run without watch mode.

## Format Code

```sh
yarn fomart
```

## Run Dev

```sh
yarn dev
```

Then open [localhost:8000/src/Main.elm](http://localhost:8000/src/Main.elm)

## Build For Production

```sh
yarn build
```

Output is in `./build/elm.min.js`.
